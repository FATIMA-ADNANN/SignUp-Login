#  Authentication System UI

A modern and responsive Authentication Interface built using HTML and CSS.

This project includes both **Sign Up** and **Login** pages with clean layout, styled inputs, and structured form design.


##  Features

-  Sign Up form
-  Login form
- Styled input fields
- Custom date picker styling
- Terms & Conditions checkbox
- Clean UI layout
- Organized folder structure


##  Built With

- HTML5
- CSS3
- Git & GitHub
